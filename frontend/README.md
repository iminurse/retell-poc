# Retell POC Frontend

React + Vite + TypeScript frontend for Retell AI phone call integration.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The app will be available at http://localhost:5173

## Features

- **Make Call Section**: Enter phone number and initiate Retell AI calls
- **Post Call Section**: Real-time call status updates and analysis display
- **Auto-polling**: Automatically polls for call status and analysis updates

## Usage

1. Enter a phone number in E.164 format (e.g., +91XXXXXXXXXX)
2. Click "Call phone number" to initiate the call
3. Monitor call status in the Post Call section
4. View call analysis once the call is completed and analyzed
